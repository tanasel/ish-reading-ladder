These instructions already include your text. Here is the whole hand-off, step by step:

  1. Press Copy instructions (below).
  2. Open the school's AI, paste into the message box and send. Open Gemini ↗ — or ChatGPT / Claude. Opens in a new tab; your text stays safe here.
  3. The AI answers with a long block that looks like computer code, full of { } brackets. That is exactly right — don't worry if it looks odd: { "schema": "ish-eal@1", "levels": [ { "phase": "Phase 1", "passage": "Water moves around the Earth…" } ] }
  4. Select all of it — click in the reply and press Ctrl/Cmd+A, then copy.
  5. Come back and paste it into step 5.

🔒 Pasting into ISH's approved AI is fine — it's the tool staff are already cleared to use. This page sends nothing itself.

5 Paste the reply

🔒 This opens once you finish step 3.
